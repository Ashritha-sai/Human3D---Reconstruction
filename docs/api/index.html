Key Classes

GaussianTrainer

Main class for training 3D Gaussians from RGB-D input. Includes methods for initialization, optimization, rendering, and export.

GaussianConfig

Dataclass for training configuration including learning rates, iterations, and densification parameters.

CameraParams

Camera intrinsic parameters including focal length, principal point, and image dimensions.

GaussianLosses

Combined loss function class with configurable weights for L1, SSIM, LPIPS, and regularization.
